    使用限制

    • 为了验证基本的请求合法性，农卷风开放服务会对请求进行 IP 地址限制，需要您提供调用端服务器 IP 地址。

    • 为平衡服务器压力，平衡调用端业务需求，农卷风开放服务对 OpenApi 各接口的调用进行了速率限制。

    IP 地址限制

    请先通过农卷风开放服务平台【控制台】-【应用配置】设置您服务器的 IP 地址。

    若 IP 地址校验不通过，服务器将发送 403 Forbidden

    HTTP/1.1 403 Forbidden

{
  "status": 403,
  "message": "IP access prohibited"
}

    速率限制

    速率限制列表

    下面是统一的用户速率限制，随着业务量和接口增加，可能会不定期调整，请及时关注。

    范围 范围标志 速率限制
    用户认证相关接口 auth 100 次/天
    智慧农业相关接口 intelli 1000 次/天 
    目前速率限制为统一限制，暂不对特定用户提供修改速率的功能。

    及时获知限制详情

    需要限制速率的接口，我们都在响应 header 中提供了相关回馈，您可根据其中的信息，及时获知您的限制详情。

    • 如果速率限制有存余，服务器将处理请求并发送具有以下标头的响应：
    标头 内容说明
    X-RateLimit-Scope 接口所属限制范围标志
    X-RateLimit-Limit 限制周期内总的可调用次数
    X-RateLimit-Remaining 限制周期内剩余可调用次数
    X-RateLimit-Reset 限制周期结束时间的 UTC 时间戳（以秒为单位） 
    HTTP/1.1 200 OK
X-RateLimit-Scope: auth
X-RateLimit-Limit: 100
X-RateLimit-Remaining: 99
X-RateLimit-Reset: 1677383251

{
  "content": {},
  "status": 200,
  "message": "Ok"
}
    • 如果速率限制存余量不足，服务器将使用响应拒绝请求并添加标头，指示客户端在发出后续请求之前应等待多长时间（以秒为单位）。
    标头 内容说明
    X-RateLimit-Scope 接口所属限制范围标志
    Retry-After 距离下次可请求剩余时间（以秒为单位） 
    HTTP/1.1 429 Too Many Requests
X-RateLimit-Scope: auth
Retry-After: 86389

{
  "status": 429,
  "message": "Too many requests. Wait for 86389 seconds."
}
